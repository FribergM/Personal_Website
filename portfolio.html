My Projects

Welcome to my portfolio! Below are some of the key projects I've worked on, each reflecting my growing skills in back-end development and web technologies:

  • Webshop Application: One of my larger projects is a dynamic webshop. This project features categories and products pulled from a database, session-based shopping cart management, and secure login functionality using Spring Security. It showcases my ability to handle database interactions, session management, and user authentication in a full-stack environment.

  • Weather Forecast Web Service: To deepen my understanding of working with external APIs, I developed a "Best Weather Tomorrow" web page. It pulls data from a weather service to provide users with information about which location will have the best weather the following day, enhancing my knowledge of API integration and web services.

  • Text-Based Games: I've built several console-based games, including a Pokémon-inspired game, as a way to practice handling relational databases. These projects helped me grasp the importance of data management and logic structure in a dynamic, game-like environment.
